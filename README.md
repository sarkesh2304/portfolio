## 🚀 Getting Started

The portfolio is built as a single static HTML file and can be served directly:

```bash
# Clone the repository
git clone https://github.com/sarkesh2304/portfolio.git
cd portfolio

# Open in browser
open portfolio.html
# or
python -m http.server 8000
# then visit http://localhost:8000
```
